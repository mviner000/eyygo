# eyymi

To install dependencies:

UBUNTU

BUILD FILES

```bash
go build
```
then
MIGRATE

```bash
go run manage.go runserver
```

RUNSERVER
```bash
go run manage.go migrate
```

START NEW APP
```bash
go run manage.go startapp
```

START SERVER by BUN

```bash
bun install
```

To run:

```bash
bun run index.ts
```
