# eyygo

To install dependencies:

UBUNTU

BUILD FILES

```bash
go build
```
then
RUNSERVER

```bash
go run manage.go runserver
```

MIGRATE
```bash
go run manage.go migrate
```

SHOW ROUTES
```bash
go run manage.go show_routes
```

EXTENDABLE PLUS INSTALLED_APP_NAME
```bash
go run manage.go show_routes app_name
```

START NEW APP
```bash
go run manage.go startapp
```
